# 🏥 AI-Powered Scheduling & Occupancy Tracker POC

![Status](https://img.shields.io/badge/Status-POC-blue) ![Tech Stack](https://img.shields.io/badge/Tech-Python_|_FastAPI_|_Supabase-success)

Welcome to the **AI-Powered Scheduling & Occupancy Tracker** for premium integrative healthcare facilities. This Proof of Concept (POC) streamlines the complex scheduling of modern biohacking therapies and gym facilities, mitigating no-shows and providing real-time occupancy insights.

---

## ✨ Key Features

*   **🤖 Conversational AI Chatbot:** Book treatments and check real-time facility crowding using natural language.
*   **🧠 Intelligent Backend Engine:** Automatically resolves scheduling constraints and proactively optimizes booking slots.
*   **📊 Real-Time Admin Dashboard:** Monitor live gym occupancy and manage appointments categorized by service type.

---

## 🛠️ Tech Stack

*   **Backend API Framework:** ⚡ FastAPI (Python)
*   **Database:** 🐘 Supabase (PostgreSQL)
*   **Frontend (Admin Dashboard):** 📈 Streamlit (Python)
*   **Frontend (Chatbot UI):** 💬 Botpress / Dialogflow
*   **AI / NLP Layer:** 🧠 OpenAI API (GPT-4o-mini)
*   **Predictive Modeling (Mock):** 🔮 Scikit-learn (Python)
